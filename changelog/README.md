# Changelog

What changed in Claude Desktop, in plain English.

Each entry corresponds to a version of the app and is auto-drafted by the [pipeline](../pipeline/README.md), then reviewed by a human before publishing.

## Format

Filenames follow `YYYY-MM-DD-vX.Y.md` (date the entry was published, app version it documents).

Each entry includes:

- **What's new** — features that didn't exist before
- **What's changed** — things that work differently
- **What's gone** — features that were removed or hidden
- **Affected chapters** — which parts of the book needed updates
