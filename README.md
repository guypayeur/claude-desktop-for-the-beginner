# Claude Desktop: For the Beginner in All of Us

An unofficial, plain-English guide to **Claude for Windows** and **Claude for Mac** — the desktop chat app from Anthropic.

Written for people who don't write code. Stays current via an LLM-in-the-loop pipeline that watches the official release notes and proposes updates whenever Anthropic ships changes.

> Not affiliated with Anthropic. "Claude" is a trademark of Anthropic PBC. This project paraphrases and reorganizes publicly available information for a beginner audience.

## How this stays fresh

Claude Desktop changes fast. A daily pipeline:

1. Watches the [official release notes](https://docs.anthropic.com/en/release-notes/claude-apps) and [Anthropic news](https://www.anthropic.com/news)
2. When it detects a change, drafts a changelog entry plus a list of suggested chapter edits
3. Opens a pull request — a human reviews, retakes screenshots if needed, and merges

See [pipeline/README.md](pipeline/README.md) for the design.
